## How It Works

- **Rubric Packs** — YAML-defined evaluation dimensions with weighted criteria
- **Scoring Engine** — matches positive/negative signals, computes weighted scores
- **Reporter** — rich terminal output with letter grades or exportable markdown
- **Benchmarks** — JSONL test suites to validate rubric accuracy
